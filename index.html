Proyectos Principales

Ballopop

Esta es una WebApp de sistema de ballotaje, con soporte completo para DNI argentino.
Escrita en java EE (arquitectura MVC), con frontend JSP + Booststrap, Driver custom JDBC, Base de datos en MySQL, y encripción de contraseñas mediante Jasypt (tanto para Votantes como para Admins)
Soporta Deployment local o en la nube mediante Apache Tomcat.

ECommerce API

API de fácil deployment en la nube, pensada para ser integrada con una vista basada en JSP (aunque soporta HTML5 directamente también)
La función de la misma es la de hacer de sistema completo para una página de ECommerce.
Construida con Java EE (arquitectura MVC), y Base de datos MySQL.

Music Player

Un Ligero reproductor de música escrito en Java SE, y con frontend basado en Swing.
Soporta archivos .wav de manera nativa, posee un completo explorador de archivos, y admite carga de álbumes enteros.
 Soporta OS basados en Linux, BSD, o MacOs.

Resistor-tool

Herramienta creada para medir las especificaciones de Resistores eléctricos. Soporta Resistores de 3, 4, 5 y 6 bandas, y puede medir Resistencia, Tolerancia, y Coeficiente térmico.
Escrita en Java SE y HTML. Utiliza componentes Swing.
Soporta OS basados en Linux, BSD, y MacOS.

Simulación de banco

Una sencilla Simulación de banco escrita en 700 líneas de código, usada como ejemplo para mostrar el potencial de la librería de GUI Swing (Java SE).
